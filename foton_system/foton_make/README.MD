## Cadastro de Objetos - Foton System

Este é um projeto em Python que tem como objetivo criar uma solução para auxiliar arquitetos urbanistas a criarem e gerenciarem um programa de necessidades de seus projetos arquitetônicos. A solução permite cadastrar, coletar informações e deletar objetos presentes em diferentes ambientes de uma residência.

### Funcionalidades

- Adicionar um objeto com suas características físicas, funções e relação com mobiliários em ambientes específicos.
- Coletar informações sobre um objeto cadastrado.
- Deletar um objeto do dicionário de objetos.
- Visualizar todos os objetos cadastrados.

### Requisitos

- Python 3.x

### Instalação

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
cd nome-do-repositorio
```

2. Instale as dependências:

```bash
pip install -r requirements.txt
```

### Uso

Execute o programa:

```bash
python fotonMake.py
```

A interface gráfica permitirá interagir com o dicionário de objetos. Você poderá adicionar novos objetos, coletar informações sobre objetos existentes, deletar objetos e visualizar todos os objetos cadastrados.
